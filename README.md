# DuploCloud Landing Page

A single-page pitch built for a conference conversation with the DuploCloud team (Venkat Thiruvengadam, Matt Amundson, and the GTM bench). The page is styled in DuploCloud's own design vocabulary, deep purple hero with the signature yellow-to-pink-to-violet gradient as a single accent, Poppins throughout, pill-shaped CTAs, and a layered-stack narrative keyed to their "IaC, platform, AI agent" framing. The pitch body argues that Duplo's AI DevOps Engineers are the 10% of the stack done right, and that the decision framework for the other 90% is the work one layer up. The CTA routes to Matt Creamer, CRO at Eduba, for a 30-minute conversation.
